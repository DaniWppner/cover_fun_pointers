
# syzkaller Smoke Test Resources

This directory contains resources to extract the QEMU commands needed to test an instrumented kernel as used by syzkaller.

## 1. Prerequisites

- A working syzkaller installation
- QEMU image generated by syzkaller scripts

## 2. Configuration

- Edit `syzkaller.cfg` to match your local paths and environment.

## 3. Run syzkaller in Smoke Test Mode

Use the following command to run syzkaller and capture the QEMU commands from the log:

```bash
/path/to/bin/syz-manager -config ./syzkaller_tmp_dir/syzkaller.cfg -debug -mode smoke-test 2>&1 | tee ./syzkaller_tmp_dir/smoke_test_log
```

It will fail because syzkaller expects a different kcov format that what we are using, wich is expected. The goal is to confirm that everything else around the compiled kernel works.

Replace `/path/to/` with your actual paths as needed.
